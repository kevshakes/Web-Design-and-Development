Hyperlinks vs Anchors — Demo

1) Hyperlinks (navigate between pages)

These links point to different pages. In a real site you'd create separate files like home.html and about.html. Clicking them loads another page (or tab).

Home (home.html)  •  About Us (about.html)

Tip: open those files in your editor or browser to see separate pages. If the files don't exist yet, create them and the links will work as normal hyperlinks.

2) Anchors (jump inside the same page)

Anchors use fragment identifiers (hashes) to jump to elements that have matching id attributes on the same page.

Intro

This section demonstrates scrolling to a location within the same document using an anchor link.

When you click a link such as #section-features the browser will scroll to the element that has id="section-features".
