CSS Examples: Inline, Embedded, External

External stylesheet

This paragraph is styled by the external stylesheet (styles.css). Create a file named styles.css beside this HTML file to see the external styles applied.

Embedded (internal) <style> block

This paragraph is styled by the embedded <style> block inside this document.

Inline style (style attribute)

This paragraph uses an inline style. Inline styles have the highest priority for the properties they set, so the inline color overrides the embedded/external color.

Precedence demo

Inline color (crimson) — overrides embedded/external color.

Embedded color (blue) — used when not overridden by inline.

External-only styling (see styles.css) — used when no embedded/inline overrides apply.

Suggested contents for styles.css (create this file next to the HTML): 
/* styles.css */
.external-box {
    background: #e8f5e9;
    border: 2px dashed #66bb6a;
    padding: 10px;
    margin-bottom: 1rem;
}
.shared {
    color: #2e7d32;
}
